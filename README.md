# Python Mini Projects

A collection of Python projects built using OOP and Turtle graphics.

## Projects
- **Snake Game** - Classic snake game with score tracking and high score persistence
- **Pong Game** - Two player pong game
- **Hangman** - Word guessing game
- **Blackjack** - Card game with game logic
- **Coffee Machine Simulator** - OOP-based coffee machine
- **Password Generator** - Random password generator

## Tech Stack
Python, Turtle Graphics, OOP

## How to Run
Each project is in its own folder. Navigate to the folder and run main.py
